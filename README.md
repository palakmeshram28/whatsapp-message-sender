# WhatsApp Message Sender using Python

This project sends scheduled WhatsApp messages using Python automation.

## Features
- Send message at a fixed time
- Uses WhatsApp Web
- Beginner-friendly Python project

## Technologies
- Python
- pywhatkit

## How to Run
1. Install requirements: `pip install pywhatkit`
2. Update phone number and time in code
3. Run the script

## Note
Use only for personal or consent-based messaging.
